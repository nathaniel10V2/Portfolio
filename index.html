Mes réalisations

🥗 Application de Suivi Nutritionnel

Contexte :
Ce projet a été réalisé dans le cadre de ma formation de développeur Java chez Aston IT. Il s'agit d'un projet d'apprentissage en groupe, autour de la conception d'une application web dédiée à la nutrition.

Objectif du projet :
Permettre aux utilisateurs de planifier leurs repas, de suivre leur apport nutritionnel, et de définir un objectif de poids. Le projet a été partiellement développé afin de mettre en pratique les technologies modernes utilisées en entreprise.

Ce que j'ai réalisé :

  • Développement du backend (APIs REST) en Spring Boot
  • Création du modèle de base de données relationnelle (MySQL)
  • Construction du frontend Angular avec formulaires et composants
  • Utilisation de Docker pour la conteneurisation du projet
  • Travail collaboratif via Git et GitHub

Technologies utilisées :
Java 17, Spring Boot 3 (JPA, MVC), Angular 16, MySQL, Docker, GitHub

Application de Gestion d'un établissement scolaire

Description du projet
Ce projet a pour objectif de faciliter la gestion quotidienne d'un établissement scolaire. Il permet de gérer efficacement les informations liées aux élèves, enseignants, cours, et notes.

Fonctionnalités principales :

  • Gestion des entités : Ajout, modification, suppression et affichage des élèves, enseignants, matières et notes
  • Affichage dynamique : Tableaux responsives, filtres, recherche et actions
  • API REST développée avec Spring Boot pour le traitement des données
  • Interface utilisateur Angular : navigation fluide, formulaires réactifs, responsive design
  • Base de données Oracle : gestion des relations complexes (ex. : élèves et notes, enseignants et matières)

Technologies utilisées :
Java 21, Spring Boot 3 (JPA, MVC), Angular 19, Oracle Database
